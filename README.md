pianoTab is a kivy based score editor that plots a midi-file in a readable way on a document. 
Futures:
- the program renders/creates a high quality vector pdf as output.
- it has a advanced point and click editor with contextual toolbar
- the interface is inspired by the design of the classical sunvox gui ([www.warmplace.ru/sunvox](https://warmplace.ru/soft/sunvox/)).
